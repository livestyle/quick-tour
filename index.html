LiveStyle

is a unique tool for live bi-directional stylesheet editing

Unlike other live edit tools, LiveStyle integrates directly into web browser and code editor to bring you the best live edit experience.
To start using LiveStyle, simply connect any CSS file from your web-page with one of the opened stylesheet files in editor.

Now all your editor changes are instantly pushed to browser.

Without file saving or page reloading.

And vice versa.

You don’t have to worry about connecting two completely different stylesheets: LiveStyle transmits updated properties only and doesn’t overwrite entire file.

Connecting browser and editor files gives you unmatched flexibility.

You can use files from your hard drive, USB drive, FTP, network mount or even new, unsaved file.
